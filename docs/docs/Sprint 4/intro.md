# Sprint 4

Durante a sprint 4 do projeto, o foco do desenvolvimento da equipe foi a integração entre todos os sistemas desenvolvidos, permitindo possuir uma versão já finalizada do MVP proposto no início do projeto. Além disso, a equipe também se propôs a fazer testes de carga em cima da infraestrutura de ingestão de dados implementada em momentos anteriores.
